APRECIEREA GLOBALA PRIN METODA ALTMAN
           Metoda Altman denumita si metoda Z se calculeaza dupa urmatoarea formula :
Z=1,2 *X1+1,4*X2+3,3*X3 +0,6*X4 + 0.999*X5 , in care:
X1 = capitalul circulant/total active
X2 = profitul net reinvestit/total active
X3 = profitul brut/total active
X4 = valoarea actiunilor de pe piata/datorii totale
X5 = cifra de afaceri/total active
      In cazul nostru cei cinci x devin:
     X1 = 162.789.534 / 260.150.328 = 0.6258
     X2 = 2.516.797 / 260.150.328 = 0.0097
     X3 = 19.723.030 / 260.150.328 = 0.0758
     X4 = 0 / 81.848.637 = 0
     X5 = 230.557.830 / 260.150.328 = 0.8862
   Altfel, functia Altman arata altfel:
Z = 1.2 * 0.6258 + 1.4*0.0097 + 3.3*0.0758 + 0.6*0 + 0.999*0.8862
Z = 0.75096 + 0.01358 + 0.25014+ 0 + 0.93051 = 1.94519
      Dupa cum se observa Z este mai mare decat 1,8 cee ace arata ca firma poate avea dificultati. Aceste dificultati se pot remedia prin micsorarea cheltuielilor, cresterea CA, dezvoltarea prduselor, identificarea unui nou segment de piata etc
